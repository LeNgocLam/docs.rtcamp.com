## clean

Clean NGINX FastCGI cache, Opcacache, Memcache

	ee clean

clean fastgi cache by default.


### Clean FastCGI cache

	ee clean --fastcgi

### Clean Memcache

	ee clean --memcache

### Clean OPcache

	ee clean --opcache

### Clean Pagespeed cache

	ee clean --pagespeed

### Clean FastCGI, Memcache, OPcache and PageSpeed cache

	ee clean --all
