### bbPress Plugin


Use following filter to remove bbPress plugin support


### Example


```php
function custom_rtp_show_bbpress_support(){
	return false;
}

add_filter( 'rtp_show_bbpress_support', 'custom_rtp_show_bbpress_support' );
```
